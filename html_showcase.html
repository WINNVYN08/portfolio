HTML

In August, I embarked on a project to create a personal portfolio website using HTML, CSS, and a bit of JavaScript as part of my NCEA assessment. The goal was to build an engaging platform to showcase my skills, projects, and achievements, making it an effective tool for both personal development and professional networking. I began by researching various portfolio layouts and decided on a clean, minimalist design that would be accessible to a wide audience. After outlining the structure, I created multiple sections, such as "About Me," and "Projects," each with its own content and styling. I used HTML to structure the page and CSS for visual enhancements, ensuring the interface was visually appealing and user-friendly. As users navigated through the portfolio, the site provided interactive elements and smooth transitions, making for an enjoyable browsing experience. This involved utilizing JavaScript for dynamic content and animations, which helped reinforce my understanding of web development concepts. Once the portfolio was complete, I tested it on different devices and browsers to ensure compatibility and responsiveness. This project was not only a practical application of the web development skills I was learning but also an excellent opportunity to enhance my design and problem-solving abilities. Overall, creating the portfolio was a rewarding experience that deepened my appreciation for web development while fulfilling the requirements of my assessment.
